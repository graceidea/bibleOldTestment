\section{陷害雅各的罪 10-14}
\begin{itemize} 
\itemsep-.5em 
\item 因你向兄弟雅各\underline{行強暴}，羞愧必遮蓋你，你也必永遠斷絕
\item 當外人擄掠雅各的財物，外邦人進入他的城門，為耶路撒冷拈鬮的日子，你竟\underline{站在一旁，像與他們同夥}
\item 你兄弟遭難的日子，你不當瞪眼看著
\item 猶大人被滅的日子，你不當因此\underline{歡樂}，他們遭難的日子，你不當\underline{說狂傲的話}
\item 我民遭災的日子，你不當\underline{進他們的城門}，他們遭災的日子，你不當\underline{瞪眼看著}他們受苦
\item 他們遭災的日子，你不當\underline{伸手搶}他們的財物，你不當站在岔路口，\underline{剪除}他們中間逃脫的
\item 他們遭難的日子，你不當將他們剩下的人\underline{交付仇敵}
\end{itemize}
