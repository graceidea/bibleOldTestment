\section{大利烏王第二年六月 1:1-15}

\subsection{六月初一日,耶和華耶和華的話 1:1-12}
\subsubsection{耶和華向所羅巴伯和約書亞說話 1:1}
\begin{itemize} 
\itemsep-.35em 
\item 大流士王第二年六月初一日
\item 耶和華的話藉先知哈該
\item 向撒拉鐵的兒子\underline{猶大省長所羅巴伯}和約薩答的兒子\underline{大祭司約書亞}說
\end{itemize}

\subsubsection{耶和華話的内容 1:2-12}
\begin{itemize} 
\itemsep-.35em 
\item 萬軍之耶和華如此說：這百姓說，建造耶和華殿的時候尚未來到。
\item 那時耶和華的話臨到先知哈該說：這殿仍然荒涼，你們自己還住天花板的房屋嗎？ 
\item 現在萬軍之耶和華如此說：你們要省察自己的行為。
\begin{itemize} 
\itemsep-.35em 
\item 你們撒的種多，收的卻少
\item 你們吃卻不得飽，喝卻不得足
\item 穿衣服卻不得暖
\item 得工錢的將工錢裝在破漏的囊中
\end{itemize}
\item 萬軍之耶和華如此說：你們要省察自己的行為
\begin{itemize} 
\itemsep-.35em 
\item 你們要上山取木料，建造這殿
\item 我就因此喜樂，且得榮耀
\end{itemize}
\item 這是耶和華說的
\begin{itemize} 
\itemsep-.35em 
\item 你們盼望多得，所得的卻少，你們收到家中，我就吹去，這是為什麼呢？
\item 因為我的殿荒涼，你們各人卻顧（奔）自己的房屋
\end{itemize}
\item 這是萬軍之耶和華說的。所以為你們的緣故，天就不降甘露，地也不出土產。我命乾旱臨到地土、山岡、五穀、新酒和油，並地上的出產、人民、牲畜，以及人手一切勞碌得來的。
\end{itemize}



\subsection{六月二十四日,百姓和官长的反应 1:13-15}
\begin{itemize} 
\itemsep-.35em 
\item 那時，撒拉鐵的兒子所羅巴伯和約薩答的兒子大祭司約書亞，並剩下的百姓，都聽從耶和華他們神的話和先知哈該奉耶和華他們神差來所說的話。百姓也在耶和華面前存敬畏的心
\item 耶和華的使者哈該奉耶和華差遣對百姓說：「耶和華說：我與你們同在。」
\item 耶和華激動猶大省長撒拉鐵的兒子所羅巴伯和約薩答的兒子大祭司約書亞，並剩下之百姓的心，他們就來為萬軍之耶和華他們神的殿做工。這是在大流士王第二年六月二十四日
\end{itemize}


\section{大利烏王七月二十一日 2:1-9}
\subsection{七月二十一日得启示 2:1-2}
\begin{itemize} 
\itemsep-.35em 
\item 七月二十一日，耶和華的話臨到先知哈該說：
\item 你要曉諭猶大省長撒拉鐵的兒子所羅巴伯和約薩答的兒子大祭司約書亞，並剩下的百姓說
\end{itemize}

\subsection{我與你們同在，你們不要懼怕 2:3-5}
\begin{itemize} 
\itemsep-.35em 
\item 你們中間存留的，有誰見過這殿從前的榮耀呢？現在你們看著如何？豈不在眼中看如無有嗎？
\item 耶和華說：所羅巴伯啊，雖然如此，你當剛強！約薩答的兒子大祭司約書亞啊，你也當剛強！這地的百姓，你們都當剛強做工！
\item 因為我與你們同在。這是萬軍之耶和華說的。這是照著你們出埃及我與你們立約的話，那時我的靈住在你們中間。你們不要懼怕！
\end{itemize}

\subsection{這殿後來的榮耀必大過先前 2:6-9}  
\begin{itemize} 
\itemsep-.35em 
\item 萬軍之耶和華如此說：過不多時，
\begin{itemize} 
\itemsep-.35em 
\item 我必再一次震動天地、滄海與旱地。
\item 我必震動萬國，萬國的珍寶必都運來（萬國所羨慕的必來到）
\item 我就使這殿滿了榮耀
\item 這是萬軍之耶和華說的 
\end{itemize}
\item 萬軍之耶和華說：
\begin{itemize} 
\itemsep-.35em 
\item 銀子是我的，金子也是我的。
\item 這殿後來的榮耀必大過先前的榮耀，
\item 在這地方我必賜平安。
\item 這是萬軍之耶和華說的。
\end{itemize}
\end{itemize}


\section{大利烏王九月二十四日 2:10-23}
\subsection{九月二十四日神第一次说话 2:10-19}
\begin{itemize} 
\itemsep-.35em 
\item 大流士王第二年九月二十四日，耶和華的話臨到先知哈該說
\item 萬軍之耶和華如此說：你要向祭司問律法
\begin{itemize} 
\itemsep-.35em 
\item （哈該說）若有人用衣襟兜聖肉，這衣襟挨著餅或湯，或酒，或油，或別的食物，便算為聖嗎？ 
\item 祭司說：「不算為聖。」 
\item 哈該又說：「若有人因摸死屍染了汙穢，然後挨著這些物的哪一樣，這物算汙穢嗎？」
\item 祭司說：「必算汙穢。」
\item 於是哈該說：「耶和華說：
\begin{itemize} 
\itemsep-.35em 
\item 這民這國在我面前也是如此，他們手下的各樣工作都是如此，他們在壇上所獻的也是如此 
\item 現在你們要追想，此日以前，耶和華的殿沒有一塊石頭壘在石頭上的光景。在那一切日子有人來到穀堆，想得二十斗只得了十斗；有人來到酒池，想得五十桶只得了二十桶。在你們手下的各樣工作上，我以旱風、霉爛、冰雹攻擊你們，你們仍不歸向我。這是耶和華說的 
\item 你們要追想此日以前，就是從這九月二十四日起，追想到立耶和華殿根基的日子。倉裡有穀種嗎？葡萄樹、無花果樹、石榴樹、橄欖樹都沒有結果子。從今日起，我必賜福於你們。」
\end{itemize}
\end{itemize}
\end{itemize}


\subsection{九月二十四日神第二次说话 2:20-23}
\begin{itemize} 
\itemsep-.35em 
\item 這月二十四日，耶和華的話二次臨到哈該說：你要告訴猶大省長所羅巴伯說
\begin{itemize} 
\itemsep-.35em 
\item 我必震動天地
\item 我必傾覆列國的寶座
\item 除滅列邦的勢力，並傾覆戰車和坐在其上的
\item 馬必跌倒，騎馬的敗落
\item 各人被弟兄的刀所殺
\end{itemize}
\item 萬軍之耶和華說：
\begin{itemize} 
\itemsep-.35em 
\item 我僕人撒拉鐵的兒子所羅巴伯啊，到那日，我必以你為印
\item 因我揀選了你。這是萬軍之耶和華說的
\end{itemize}
\end{itemize}
