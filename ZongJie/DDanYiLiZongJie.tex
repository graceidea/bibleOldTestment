

\section{但以理書总结}
\subsection{但以理書中的赞美}
\subsubsection{但以理的赞美 2:20-23}
「神的名是應當稱頌的，從亙古直到永遠！因為智慧能力都屬乎他。 
21 他改變時候、日期，廢王、立王，將智慧賜予智慧人，將知識賜予聰明人。 
22 他顯明深奧隱祕的事，知道暗中所有的，光明也與他同居。 
23 我列祖的神啊，我感謝你，讚美你！因你將智慧、才能賜給我，允准我們所求的，把王的事給我們指明。」
\begin{itemize}
\itemsep-.45em  
\item 神的名是應當稱頌的，從亙古直到永遠
\item 智慧能力都屬乎他
\item 將智慧賜予智慧人，將知識賜予聰明人
\item 他顯明深奧隱祕的事
\item 將智慧、才能賜給我，允准我們所求的
\end{itemize}

\subsubsection{尼布甲尼撒王的赞美 2:47}
\begin{itemize}
\itemsep-.45em  
\item 你們的神使你(但以理)能顯明這奧祕的事，
\item 你們的神誠然是萬神之神，萬王之主，
\item 又是顯明奧祕事的
\end{itemize}

\subsubsection{尼布甲尼撒王的赞美 3:28-29}
「沙得拉、米煞、亞伯尼歌的神是應當稱頌的！
他差遣使者救護倚靠他的僕人，他們不遵王命，捨去己身，在他們神以外不肯侍奉敬拜別神。 
29 現在我降旨，無論何方、何國、何族的人，謗讟沙得拉、米煞、亞伯尼歌之神的，必被凌遲，他的房屋必成糞堆，因為沒有別神能這樣施行拯救。」
\begin{itemize}
\itemsep-.45em  
\item 沙得拉、米煞、亞伯尼歌的神是\underline{應當稱頌}的
\item 他\underline{差遣使者救護倚靠他的僕人}
\item 我降旨，無論何人都\underline{不得謗讟沙得拉、米煞、亞伯尼歌之神}
\item 因為\underline{沒有別神}能這樣施行拯救
\end{itemize}


\subsubsection{尼布甲尼撒王的赞美 4}
\subsubsection{谕旨开头的赞美 4:2-3}
\begin{itemize}
\itemsep-.45em  
\item 我樂意將至高的神向我所行的神蹟奇事宣揚出來
\item 他的神蹟何其大，他的奇事何其盛！
\item 他的國是永遠的，他的權柄存到萬代！
\end{itemize}

\subsubsection{谕旨结尾的赞美 4:34-37}
日子滿足，我尼布甲尼撒舉目望天，我的聰明復歸於我，我便稱頌至高者，讚美、尊敬活到永遠的神。他的權柄是永有的，他的國存到萬代。 
35 世上所有的居民都算為虛無，在天上的萬軍和世上的居民中，他都憑自己的意旨行事。無人能攔住他手，或問他說：「你做什麼呢？」 
36 那時，我的聰明復歸於我，為我國的榮耀，威嚴和光耀也都復歸於我，並且我的謀士和大臣也來朝見我。我又得堅立在國位上，至大的權柄加增於我。 
37 現在我尼布甲尼撒讚美、尊崇、恭敬天上的王，因為他所做的全都誠實，他所行的也都公平，那行動驕傲的，他能降為卑。
\begin{itemize}
\itemsep-.45em  
\item 日子滿足，预言应验
\begin{itemize}
\itemsep-.45em  
\item 我尼布甲尼撒舉目望天，我的聰明復歸於我
\item 我國的榮耀，威嚴和光耀也都復歸於我，
\item 並且我的謀士和大臣也來朝見我。
\item 我又得堅立在國位上，至大的權柄加增於我。
\end{itemize}
我便稱頌至高者，讚美、尊敬活到永遠的神。
\begin{itemize}
\itemsep-.45em  
\item 他的權柄是永有的，他的國存到萬代。 
\item 他都憑自己的意旨行事。無人能攔住他手
\item 他所做的全都誠實，他所行的也都公平
\item 那行動驕傲的，他能降為卑。
\end{itemize}
\end{itemize}

\subsubsection{大利乌王的赞美 6:26-27}
那時大流士王傳旨，曉諭住在全地，各方、各國、各族的人說：「願你們大享平安！ 
26 現在我降旨曉諭我所統轄的全國人民，要在但以理的神面前戰兢恐懼。因為他是永遠長存的活神，他的國永不敗壞，他的權柄永存無極。 
27 他護庇人，搭救人，在天上地下施行神蹟奇事，救了但以理脫離獅子的口。」
\begin{itemize}
\itemsep-.45em

\item 他護庇人，搭救人，在天上地下施行神蹟奇事，救了但以理脫離獅子的口
\end{itemize}
